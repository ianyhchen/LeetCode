# LeetCode 84. Largest Rectangle in Histogram 筆記

## 1. 題目解析
在一個柱狀圖陣列中，尋找可以勾勒出的最大矩形面積。每個柱子寬度為 1，矩形高度受限於範圍內最矮的柱子。

## 2. 核心演算法與資料結構
* **資料結構**：單調遞增棧 (Monotonic Increasing Stack)。
* **哨兵技巧**：在原陣列前後各加一個高度為 0 的柱子，用以簡化邊界處理（確保棧不為空且最後能清空棧）。

## 3. 解題思路與方法比較
* **核心思路**：枚舉每一根柱子 $i$，將其視為矩形的「高度瓶頸」，尋找其左邊與右邊第一個比它矮的邊界 $L$ 與 $R$。
* **方法比較**：
    * 暴力法：$O(n^2)$，會 TLE。
    * 單調棧：$O(n)$，遍歷一次陣列，每個元素進出棧各一次，效率最高。

## 4. 常見錯誤與陷阱
* **While 迴圈缺失**：當前柱子可能比棧內多個元素矮，必須使用 `while` 連續彈出並計算面積。
* **寬度計算**：彈出 `mid` 後，左邊界是 `stack[-1]`，右邊界是當前索引 `i`，寬度公式為 $i - stack[-1] - 1$。
* **索引混淆**：棧裡面存的是「索引」而非「高度」，計算面積時需透過 `heights[mid]` 取值。
