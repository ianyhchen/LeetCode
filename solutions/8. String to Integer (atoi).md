# LeetCode 8. String to Integer (atoi) - Python 實作筆記

## 1. 題目解析
將字串轉換為 32 位元有號整數，需嚴格處理前導空白、正負號、非數字中斷以及 32-bit 溢位截斷。

## 2. 核心演算法與資料結構
- **演算法**：線性掃描 (Linear Scan)
- **空間複雜度**：$O(1)$，僅使用固定變數。
- **時間複雜度**：$O(N)$，字串僅掃描一次。

## 3. 解題思路
採用**三段式處理**：
1. **Skip Whitespaces**: 移動指標 `i` 跳過開頭所有 `' '`。
2. **Handle Sign**: 檢查 `s[i]` 是否為 `'+'` 或 `'-'`，更新 `sign` 並將指標 `i` 後移。
3. **Convert Digits**: 
   - 遍歷剩餘字元，遇到非數字即停止。
   - **溢位檢查**：在 `res * 10` 之前，先檢查當前 `res` 是否大於 `214748364`。若等於該值且下一位 `digit > 7`，則判定溢位，依正負號回傳邊界。

## 5. 常見錯誤與陷阱
- **Index Out of Range**: 必須在存取 `s[i]` 前始終檢查 `i < n`。
- **32-bit 邊界**: 正數最大值尾數是 7，負數最小值絕對值尾數是 8，若使用正數處理累加，統一判斷 `> 7` 即可安全攔截。
- **無效輸入**: 字串開頭非數字或符號時應回傳 0。
