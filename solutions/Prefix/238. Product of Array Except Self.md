# LeetCode 238. Product of Array Except Self 筆記

## 1. 題目解析
* **目標**：給定一個整數陣列 `nums`，回傳一個新陣列 `answer`，其中 `answer[i]` 等於 `nums` 中除了 `nums[i]` 以外所有元素的乘積。
* **核心限制**：
    * **時間複雜度**：必須為 $O(n)$。
    * **禁止使用除法**：不能先算總乘積再除以各個元素（這也避開了除以 0 的風險）。
    * **進階要求**：空間複雜度需為 $O(1)$（輸出陣列不計入空間）。

---

## 2. 核心演算法與資料結構
* **核心思想**：**前綴乘積 (Prefix Product)** 與 **後綴乘積 (Suffix Product)**。
* **數學原理**：
    對於陣列中的任何位置 $i$：
    $$\text{Answer}[i] = (\text{Prefix Product up to } i-1) \times (\text{Suffix Product from } i+1)$$
* **資料結構**：陣列 (Array)。

---

## 3. 解題思路與方法比較

| 特性 | 方法 A：左右乘積陣列法 | 方法 B：空間優化法 (推薦) |
| :--- | :--- | :--- |
| **邏輯** | 建立 `L` 與 `R` 兩個額外陣列分別儲存前後綴積。 | 利用輸出陣列儲存前綴積，再用變數動態計算後綴積。 |
| **時間複雜度** | $O(n)$ | $O(n)$ |
| **空間複雜度** | $O(n)$ (額外開銷) | $O(1)$ (僅使用輸出陣列與常數變數) |
| **優點** | 邏輯最直觀，易於調試與理解。 | 符合面試最優解要求，節省記憶體。 |

---

## 4. 常見錯誤與陷阱
* **0 的處理**：
    * 若陣列中有一個 `0`：只有該 `0` 的位置會有非零值，其餘皆為 `0`。
    * 若陣列中有兩個以上的 `0`：所有位置的結果皆為 `0`。
* **邊界初始化**：
    * 第一個元素左邊沒有數字，其**前綴積初始值應為 1**。
    * 最後一個元素右邊沒有數字，其**後綴積初始值應為 1**。
* **整數溢出 (Overflow)**：
    * 雖然 Python 會自動處理大整數，但在 C++/Java 中需注意乘積是否超過 `int` 或 `long` 的範圍。

---

## 5. 實作技巧總結 (以空間優化法為例)
1.  **第一遍掃描 (左到右)**：
    * 令 `ans[i]` 等於 `ans[i-1] * nums[i-1]`。
    * 這步完成後，`ans` 陣列存的是每個位置的「左側積」。
2.  **第二遍掃描 (右到左)**：
    * 維護一個變數 `R = 1`（代表右側累積積）。
    * 更新 `ans[i] = ans[i] * R`。
    * 更新 `R = R * nums[i]`，供下一個（左邊的）位置使用。
