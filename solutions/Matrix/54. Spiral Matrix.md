# LeetCode 54. Spiral Matrix 刷題筆記

## 1. 題目解析
* **目標**：給定一個 $m \times n$ 矩陣，按照 **「順時針螺旋」** (Clockwise Spiral) 的順序回傳所有元素。
* **路徑規律**：由左至右 $\to$ 由上至下 $\to$ 由右至左 $\to$ 由下至上。
* **核心挑戰**：在非正方形矩陣中，當邊界收縮到最後時，需避免重複訪問已處理的行或列。

---

## 2. 核心演算法與資料結構
* **演算法**：模擬 (Simulation) / 邊界收縮法 (Boundary Shrinkage)。
* **資料結構**：2D Array (Matrix)。
* **時間複雜度**：$O(M \times N)$，矩陣中每個元素僅被訪問一次。
* **空間複雜度**：$O(1)$，除了結果輸出外，僅需常數個變數記錄邊界。

---

## 3. 解題思路與方法比較
### 邊界收縮法 (四指針控制)
透過維護 `top`, `bottom`, `left`, `right` 四個邊界，每完成一個方向的遍歷，就將該邊界向內推動。

| 動作順序 | 固定索引 | 遍歷範圍 (Python Range) | 邊界更新 |
| :--- | :--- | :--- | :--- |
| **1. 向右** | `matrix[top][c]` | `range(left, right + 1)` | `top += 1` |
| **2. 向下** | `matrix[r][right]` | `range(top, bottom + 1)` | `right -= 1` |
| **3. 向左** | `matrix[bottom][c]` | `range(right, left - 1, -1)` | `bottom -= 1` |
| **4. 向上** | `matrix[r][left]` | `range(bottom, top - 1, -1)` | `left += 1` |

**關鍵邏輯：**
在執行步驟 3 (左) 與步驟 4 (上) 之前，必須再次檢查 `top <= bottom` 與 `left <= right`。這是因為步驟 1 與 2 已經修改了邊界值，若不檢查，在扁平或瘦高矩陣中會導致重複存取。

---

## 5. 常見錯誤與陷阱
1. **Index 越界 (Off-by-one)**：
   * Python 的 `range` 是左閉右開。反向遍歷 `right` 到 `left` 必須寫成 `range(right, left - 1, -1)` 才能包含到 `left`。
2. **非正方形矩陣的重複走位**：
   * 這是本題最容易出錯的地方。若 `top` 增加後已經超過 `bottom`，則不應再執行「向左走」的動作。
3. **矩陣維度理解錯誤**：
   * `matrix[row][col]`：第一個中括號是「哪一列（上下）」，第二個是「哪一行（左右）」。
